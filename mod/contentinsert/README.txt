How to work:
1, Download the plugin, unzip and upload to mod directory.
2, Activate the plugin.
3, Deactivate HTMLawed Plugin, Because it will filter your javascript so that your advertisement code can not work well.
4, Insert your content such as adsense code to the position that you want to show to others.
5, Save your setting, and go to site check if it works well.
6, Once the advertisement shows, Activate HTMLawed plugin again.
7, Enjoy!
